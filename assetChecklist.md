General Branding Assets
brand-logo.png – 1024×1024 – PNG – Master logo


Aspect ratio can vary; just ensure it's at least this large.


brand-logo-square.png – 1024×1024 – PNG – Square version


Used when a perfect square is required (icons, avatars, etc.).


brand-logo-horizontal.png – 1600×600 – PNG/SVG – Horizontal logo


Best for website headers, emails, and wide layouts.


app-icon-master.png – 2048×2048 – PNG – Source file for all app icons


Highest‑resolution version; designer should export all icons from this.


Export the following from app-icon-master.png
Mobile App Icons
icon.png – 1024×1024 – PNG – Main Expo app icon


Expo uses this to automatically generate all iOS & Android icon sizes.


ios-appstore.png – 1024×1024 – PNG – App Store listing


Same artwork as icon.png; naming matters because Apple requires a manual upload.


android-playstore.png – 512×512 – PNG – Play Store listing


Google Play requires this separate upload.


Splash Screens
splash.png – 720×1280 or 1242×2436 – PNG – Main splash image


Keep logo centered; avoid placing text or details near edges.


splash-background.png – 2048×2048 – PNG – Resize‑safe background


Solid color or simple texture used behind the centered image when Expo scales it.


Web Favicons & PWA Icons
favicon.ico – 48×48 – ICO – Browser tab icon


Multi-resolution icon file used by desktop browsers.


favicon-32.png – 32×32 – PNG – Browser tab


Standard favicon resolution.


favicon-16.png – 16×16 – PNG – Small browser tab/icon


Often used for low-res environments.


apple-touch-icon.png – 180×180 – PNG – iOS home screen icon


Used when a user bookmarks the site on iOS.


manifest-icon-512.png – 512×512 – PNG – PWA installation icon


Primary icon used for installed web apps.


manifest-icon-192.png – 192×192 – PNG – PWA installation icon


Required fallback size for PWAs.


mask-icon.svg – SVG – Safari pinned tab icon


Monochrome icon used by macOS Safari.


Social Media Preview Images
home-preview.png – 1200×630 – PNG/JPG – OG & Twitter card


Default image shown when the website is shared.


twitter-preview.png – 1200×600 – PNG/JPG – Twitter-specific preview


Slightly different ratio optimized for Twitter cards.


share-square.png – 1080×1080 – PNG – Instagram post


Standard square share graphic.


share-story.png – 1080×1920 – PNG – Instagram story


Vertical format optimized for stories.


PWA Files
manifest.json – PWA settings


Defines app name, theme colors, and which PWA icons to use.


service-worker.js – Offline support


Enables caching and offline functionality.
